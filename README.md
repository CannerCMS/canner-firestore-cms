## Prerequisite

```sh
$ npm i -g firebase-tools
```

## Installation

```sh
$ yarn
```

## Development

```sh
yarn start
```

## Production

```sh
yarn build
```

## Serve with firebase

```sh
yarn serve
# same as
# firebase serve
```

## Deploy to firebase

```sh
yarn deploy
# same as
# firebase deploy --only hosting
```
